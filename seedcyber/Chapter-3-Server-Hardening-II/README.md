# Chapter 3: Server Hardening II – System & Service Security

**Duration:** Week 3  
**Prerequisites:** Chapter 2 complete  

## Overview

Secure the services running on your server. Deploy HTTPS, configure web servers, and disable unnecessary services.

## Objectives

✓ Deploy Nginx web server securely  
✓ Set up HTTPS with Let's Encrypt certificates  
✓ Configure DNS and domains  
✓ Harden web server configuration  
✓ Audit and disable unnecessary services  
✓ Implement logging and monitoring

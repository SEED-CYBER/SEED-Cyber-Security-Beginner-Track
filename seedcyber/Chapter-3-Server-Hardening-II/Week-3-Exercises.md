# Week 3 Exercises

## Exercise 1: DNS Setup
1. Register or use existing domain
2. Configure DNS A record to your VPS
3. Wait for propagation (nslookup verify)
4. Document setup

## Exercise 2: Nginx Installation
1. Install Nginx on your VPS
2. Create index.html in public_html
3. Configure virtual host for your port
4. Test access from local machine

## Exercise 3: HTTPS Certificate
1. Install Certbot
2. Get Let's Encrypt certificate
3. Configure Nginx for HTTPS
4. Access site via https://
5. Verify certificate validity

## Exercise 4: Service Auditing
1. List all running services
2. Disable unnecessary services
3. Configure logging
4. View access and error logs
5. Document exposed services

## Exercise 5: Security Documentation
Create report:
1. SSL/TLS certificate details
2. Nginx security headers
3. Service exposure analysis
4. Logging strategy
5. Hardened web stack

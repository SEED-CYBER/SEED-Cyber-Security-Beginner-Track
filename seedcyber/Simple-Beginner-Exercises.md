# Simple Pre-Technical Exercises for Beginners

## 1. Basic Computer Navigation
- Open a terminal (Linux) or command prompt (Windows).
- Create, rename, and delete folders and files.
- Move files between folders.

## 2. Introduction to Linux
- Boot into a Linux system (real or virtual machine).
- Use commands like `ls`, `cd`, `pwd`, `mkdir`, `rm`, and `cp`.
- Find out the current user with `whoami`.

## 3. Understanding the Internet
- Open a web browser and visit a website.
- Find your computer's IP address.
- Use `ping` to test connectivity to a website (e.g., `ping google.com`).

---

These exercises are designed to build foundational skills and confidence before starting more technical cybersecurity labs and projects.

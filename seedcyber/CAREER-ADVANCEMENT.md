# Career Advancement & Certification Guide

After completing the seedcyber **6-Week Server Security & Offensive Security program**, you have foundational knowledge across defensive and offensive security practices. This guide outlines your career options and recommended certifications.

---

## Career Pathways

### 1. Blue Team (Defensive Security)

**Focused on:** Protection, detection, incident response

**Typical Roles:**

- **Security Operations Center (SOC) Analyst**
  - Monitor security alerts 24/7
  - Investigate suspicious activities
  - Escalate incidents appropriately
  - Entry-level: $50,000 - $70,000/year
  - Experience: 1-3 years → $70,000 - $100,000+

- **Security Systems Administrator**
  - Manage firewalls, VPNs, intrusion detection systems
  - Deploy and maintain security infrastructure
  - Handle patch management and updates
  - Salary: $65,000 - $95,000/year

- **Incident Response Specialist**
  - Respond to security incidents
  - Perform forensic investigation
  - Document findings and remediation
  - Salary: $80,000 - $120,000/year

- **Compliance Officer / Security Analyst**
  - Ensure compliance with security standards (PCI-DSS, HIPAA, GDPR)
  - Conduct internal audits
  - Policy development
  - Salary: $70,000 - $110,000/year

**Recommended Certifications (Blue Team Path):**
1. ✅ Security+ (CompTIA) - Entry level foundational
2. ✅ Certified Ethical Hacker (CEH) - Practical hacking skills
3. CySA+ (CompTIA) - Advanced threat analysis
4. CCNA Security - Advanced networking security
5. CISSP - Advanced security professional (requires experience)

---

### 2. Red Team (Offensive Security)

**Focused on:** Attack, exploitation, penetration testing

**Typical Roles:**

- **Penetration Tester**
  - Conduct authorized security assessments
  - Identify vulnerabilities
  - Create detailed reports with remediation
  - Entry-level (basic): $60,000 - $80,000/year
  - Experienced: $90,000 - $150,000/year

- **Ethical Hacker / Security Researcher**
  - Develop new exploitation techniques
  - Discover zero-day vulnerabilities
  - Research security threats
  - Salary: $80,000 - $150,000/year

- **Application Security Engineer**
  - Review code for vulnerabilities
  - Perform web application testing
  - Develop secure coding practices
  - Salary: $85,000 - $130,000/year

- **Red Team Operator**
  - Simulate advanced attackers
  - Breach security systems (authorized)
  - Test incident response procedures
  - Salary: $100,000 - $180,000/year (highly specialized)

**Recommended Certifications (Red Team Path):**
1. ✅ Certified Ethical Hacker (CEH) - Practical hacking
2. ✅ eJPT (eLearnSecurity Junior Penetration Tester) - Entry level
3. OSCP (Offensive Security Certified Professional) - Industry gold standard
4. GPEN (GIAC Penetration Tester) - Advanced pentesting
5. GWAPT (GIAC Web Application Penetration Tester) - Web security specialist

---

### 3. Hybrid Roles

**Focused on:** Both offense and defense

**Typical Roles:**

- **Security Architect**
  - Design secure systems and networks
  - Risk assessment
  - Policy and standards development
  - Salary: $120,000 - $200,000/year (requires 5-10 years experience)

- **Vulnerability Manager**
  - Manage vulnerability assessment programs
  - Coordinate remediation efforts
  - Risk prioritization
  - Salary: $85,000 - $130,000/year

- **Threat Intelligence Analyst**
  - Monitor threat landscape
  - Analyze malware and attacks
  - Develop detection signatures
  - Salary: $80,000 - $130,000/year

- **Security Engineer**
  - Implement security solutions
  - Test security controls
  - Research new threats
  - Salary: $90,000 - $140,000/year

## Certification Roadmap

### Tier 1: Foundational (0-1 year experience)

```
CompTIA Security+
├─ Duration: 6-12 weeks of study
├─ Cost: $165 exam + study materials ($300-500)
├─ Difficulty: ⭐⭐ Moderate
├─ Prerequisites: None (but A+ or Network+ helpful)
└─ Value: ✅ Required for many government jobs

eLearnSecurity eJPT
├─ Duration: 4-6 weeks of study
├─ Cost: $500 (training + exam)
├─ Difficulty: ⭐ Easy-Moderate
├─ Prerequisites: None
└─ Value: ✅ Good stepping stone to OSCP
```

### Tier 2: Advanced (1-3 years experience)

```
Certified Ethical Hacker (CEH)
├─ Duration: 3-6 months study
├─ Cost: $1,200 (exam + training)
├─ Difficulty: ⭐⭐⭐ Moderate-Advanced
├─ Prerequisites: Security+ or equivalent
└─ Value: ✅ Well-recognized hacking certification

Offensive Security OSCP
├─ Duration: 2-6 months (demanding)
├─ Cost: $999 (30-day lab access + exam)
├─ Difficulty: ⭐⭐⭐⭐⭐ Very Hard
├─ Prerequisites: Practical hacking experience
└─ Value: ✅✅✅ Industry gold standard - highly valued
```

### Tier 3: Specialist (3-5+ years experience)

```
GIAC Penetration Tester (GPEN)
├─ Duration: 2-3 months
├─ Cost: $2,295 (training + exam)
├─ Difficulty: ⭐⭐⭐⭐ Hard
└─ Value: ✅ Advanced pentesting credential

CompTIA Pentest+
├─ Duration: 8-12 weeks
├─ Cost: $165 exam + materials
├─ Difficulty: ⭐⭐⭐ Moderate-Advanced
└─ Value: ✅ Good alternative to GPEN

CISSP (Certified Information Systems Security Professional)
├─ Duration: 6-12 months preparation
├─ Cost: $749 exam + training
├─ Difficulty: ⭐⭐⭐⭐ Hard
├─ Prerequisites: 5 years in 2+ security domains
└─ Value: ✅✅ Senior security professional credential
```

## Recommended Learning Path for New Graduates

### Month 1-2: Prepare for Security+
- Study for CompTIA Security+
- Hands-on practice with lab exercises
- Exam preparation
- **Cost: $400-600**
- **Outcome: Security+ Certification**

### Month 3-4: Gain Practical Skills
- Pursue eJPT or complete HackTheBox challenge
- Practice penetration testing techniques
- Build portfolio with completed challenges
- **Cost: $900-1,200**
- **Outcome: eJPT or practical portfolio**

### Month 5-9: Advanced Hacking Training
- CEH course or Security+/eJPT equivalency path
- Pursue additional practical certifications
- Build professional pentesting portfolio
- **Cost: $1,500-2,500**
- **Outcome: CEH or other advanced cert**

### Month 10-18: OSCP Pursuit (Optional but highly recommended)
- Enroll in OSCP 30-day lab
- Complete practice machines (50+ in labs)
- Attempt OSCP exam (24-hour hands-on)
- **Cost: $1,000-2,000**
- **Outcome: OSCP certification (or advanced skills)**

**Total First Year Investment: $4,000-6,000**  
**Expected Salary Increase: $15,000-30,000/year**

## Job Search Strategy

### Before Applying

1. **Complete at least Security+ or equivalent**
   - Most employers require entry-level cert

2. **Build hands-on portfolio**
   - HackTheBox machines completed
   - CTF challenges solved
   - GitHub projects showcasing skills

3. **Practice interview questions**
   - Technical security questions
   - System design
   - Behavioral questions

### Where to Find Jobs

**Job Boards:**
1. LinkedIn (most active for cyber jobs)
2. CybersecurityJobHunter.com
3. SecurityJobs.net
4. Indeed.com
5. Dice.com (tech/security focused)

**Company Targeting:**
- Cybersecurity firms (rapid growth, high salaries)
- Consulting firms (diverse projects)
- Banking/Financial (high security budgets)
- Tech companies (competitive salaries)
- Government agencies (first jobs, stability)
- In-house corporate security teams

**Networking:**
- Security conferences (DEFCON, Black Hat, DEF CON, etc.)
- Local security meetups
- LinkedIn connections
- GitHub interactions
- Twitter security community (#InfoSec)

### Interview Preparation

**Technical Interview Topics:**
- OSI Model and networking basics
- Linux command line
- Common vulnerabilities (OWASP Top 10)
- Incident response procedures
- Encryption and cryptography basics
- Network security fundamentals

**Behavioral Topics:**
- Tell me about a challenging technical problem
- How do you stay updated on security threats?
- Describe your approach to problem solving
- How do you handle working under pressure?
- Biggest security failure and what you learned

**Case Study Questions:**
- You discover a data breach. How do you respond?
- Design a secure network architecture
- How would you test an e-commerce application?
- What reconnaissance tools would you use and why?

## Salary Expectations

### Entry-Level ($50,000 - $80,000)
- Security+ required
- 0-2 years experience
- Roles: SOC Analyst, Junior Pentester, Jr. Security Admin

### Mid-Level ($80,000 - $120,000)
- CEH, CySA+ or OSCP recommended
- 2-5 years experience
- Roles: Senior SOC Analyst, Pentester, Security Engineer

### Senior ($120,000 - $180,000+)
- Multiple advanced certifications
- 5-10+ years experience
- Roles: Security Architect, Senior Pentester, CISO (in large orgs)

**Note:** Salary varies significantly by:
- Geographic location (Silicon Valley/San Francisco pays 30-50% more)
- Company size (large tech companies pay more than mid-size)
- Industry (finance/tech pays more than other sectors)
- Certifications held (each major cert adds $5,000-15,000/year)
- Consulting vs. full-time (consulting often pays 20-30% more)

## 5-Year Career Plan Example

### Year 1
- **Goal:** Secure first security role (SOC Analyst or Junior Pentester)
- **Certification:** Security+, eJPT
- **Skills Focus:** Alert monitoring, basic exploitation, networking
- **Salary:** $55,000-75,000
- **Actions:**
  - Pass Security+ exam (month 2-3)
  - Secure SOC analyst position
  - Complete 50+ HackTheBox machines
  - Get hands-on experience with enterprise tools

### Year 2
- **Goal:** Progress to mid-level analyst role
- **Certification:** CEH, CySA+ OR Pentest+
- **Skills Focus:** Advanced threat detection, web app testing
- **Salary:** $75,000-95,000
- **Actions:**
  - Earn CEH or CySA+ certification
  - Promote to Senior SOC Analyst or Pentester
  - Specialize in web application security or threat hunting
  - Contribute to or lead incident response

### Year 3-4
- **Goal:** Senior technical security role
- **Certification:** OSCP (if pursuing red team) or GPEN
- **Skills Focus:** Advanced exploitation, architecture design
- **Salary:** $100,000-130,000
- **Actions:**
  - Earn OSCP (highly valuable credential)
  - Lead penetration tests or SOC team
  - Develop new detection rules
  - Mentor junior staff

### Year 5+
- **Goal:** Leadership or specialist role
- **Certification:** CISSP (if management track) or specialized certs
- **Skills Focus:** Strategic planning, risk management
- **Salary:** $130,000-180,000+
- **Actions:**
  - Move to security architect/manager role
  - Pursue CISSP for management track
  - Manage security teams
  - Contribute to security strategy

## Conclusion

The seedcyber program has given you **solid foundational knowledge** in both defensive and offensive security. Your next steps depend on your interests:

- **Prefer defense?** → Focus on Security+, CySA+, incident response skills
- **Prefer offense?** → Pursue eJPT, CEH, OSCP certification path
- **Want both?** → Follow the hybrid path with Security+, CEH, then specialization

The **most important factors** for career advancement:
1. **Get certified** (Security+ minimum for most jobs)
2. **Build portfolio** (HackTheBox, CTF participation, GitHub projects)
3. **Network actively** (LinkedIn, conferences, local meetups)
4. **Keep learning** (new threats emerge constantly)
5. **Gain practical experience** (hands-on skills matter most)

**Good luck in your security career!**

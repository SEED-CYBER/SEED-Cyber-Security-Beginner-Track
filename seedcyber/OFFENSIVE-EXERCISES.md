# Chapter 5 & 6: Offensive Security Exercises

## Chapter 5: Week 5 Exercises

**Exercise 1: Network Reconnaissance Report**
- Scan your assigned VPS
- Nmap full port scan
- Service enumeration
- Document findings

**Exercise 2: Vulnerability Assessment**
- Run vulnerability scanner
- Document discovered weaknesses
- Assign CVE scores
- Prioritize by severity

**Exercise 3: Penetration Test Report**
- Executive summary
- Vulnerability findings
- Proof of concept (if applicable)
- Remediation recommendations
